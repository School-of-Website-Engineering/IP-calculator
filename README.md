## IP calculator
## IP计算器  
一个方便的IP计算器，只需输入IP地址即可算出该IP的掩码、前缀、判断几类网段。更多功能请进入 [LiveDemo](https://school-of-automation-engineering.github.io/IP-calculator/index.html) 使用后方可得知。
